FetchV 是一款專業通用的瀏覽器影片下載擴充功能。它相容於 Chrome、Edge 及其他以 Chromium為基礎的瀏覽器。使用 FetchV,您可以下載各種類型的線上影片,包括 HLS 流媒體影片(m3u8)、mp4、webm、flv 等。
